# Computador, Hardware e Software

## Hardware

&emsp;&emsp; - O **hardware** é um termo técnico que se refere à parte física de computadores.
&emsp;&emsp; - Um **periférico** é "um dispositivo auxiliar usado para enviar ou receber informações de um computador".

## Software 

1. O que é **Software**?

&emsp;&emsp; - Um conjunto de instruções que permite controlar um aparelho eletrônico.
&emsp;&emsp; - Parte Lógica do computador.

2. **Firmware**

&emsp;&emsp; - A função do Firmware é armazenar todas as informações para que o equipamento funcione corretamente.
&emsp;&emsp; - O firmware é armazenado permanentemente num circuito integrado (chip) de memória de memória no hardware, no momento da fabriação do componente.

3. **Drivers**

&emsp;&emsp; - A função do Driver é ser a ponte entre o Sistema Operacional e o firmaware do dispositivos eletrônicos.
&emsp;&emsp; - O Driver é instalado no computador quando há a necessidade de se conectar com um novo dispositivo.

4. **Sistema Operacional**

&emsp;&emsp; - O Sistema Operacional é responsável pelo gerenciamento de todo o hardware de seu computador.
&emsp;&emsp; - É ele que vai dizer, por exemplo, para o programa em execução, quando poderá utilizar o processador e por quanto tempo, quanto de memória RAM será usada, gerenciamento de SSD, etc.

5. Programas

&emsp;&emsp; - Software -> Programa de Computador.
&emsp;&emsp; - Programa -> Programa de Computador ou aplicativo.
&emsp;&emsp;&emsp;&emsp; - É um conjunto de instruções que descrevem uma tarefa e/ou trabalho específico do computador.
&emsp;&emsp;&emsp;&emsp; - São ferramentas desenhadas para o usuário realizar ações.

6. **Memória RAM**

&emsp;&emsp; A Memória RAM é um espaço temporário de informações do sistema operacinal e de aplicativos em uso.
&emsp;&emsp; Quando uma tarefa é concluída por algum software, é concluída por algum software, os arquivos que estavam em uso são movidos da memória RAM o HD ou SSD para serem mantidas.
&emsp;&emsp; Quando o computador é desligado, a memória é limpa automáticamente.

7. **HD/SSD**

&emsp;&emsp; O HD e o SSD são dispositivos de armazenamento de dados.
&emsp;&emsp; O HD usa pratos mecânicos e um cabeçote móvel de leitura/gravação para acessar dados.
&emsp;&emsp; O SSD não possio partes móveis, apenas chips de memória.
&emsp;&emsp; Independentemente se o computador está desligado ou não, as informações contidas no HD ou SSD permanencem guardadas.

8. **Processador**

&emsp;&emsp; - O processador é o cérebro do computador
&emsp;&emsp; - Transforma informações em uma linguagem que o computador entende -> 010111001
&emsp;&emsp; - Tem o formato de um chip.
&emsp;&emsp; - A velocidade que seu computador abre ou executa programas é em parte responsabilidade do processador
